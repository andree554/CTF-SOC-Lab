// LESSON 01 — WHAT IS AN OPEN PORT?

Think of a computer like an apartment building. The IP address is the street address. Inside there are 65,535 numbered doors — these are ports. An OPEN port has a service listening behind it. A CLOSED port slams shut. A FILTERED port is blocked by a firewall — it doesn't even answer.

// LESSON 02 — SMB: PORT 139 vs PORT 445
PORT 139 — OLD 👴
NetBIOS over TCP from the 1980s. Slow, chatty legacy protocol. Still present on unpatched Windows for backwards compatibility.
PORT 445 — MODERN ✅
SMB directly over TCP/IP — no NetBIOS middleman. Introduced with Windows 2000. This is the port nmap shows on every modern machine.
🍌 DAVE SAYS: "We target 445 because it's SMB speaking directly — no translator needed. 139 is the janitor's back entrance from 1987."
